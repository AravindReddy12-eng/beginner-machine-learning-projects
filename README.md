# Beginner Machine Learning Projects

This repository contains a collection of supervised learning projects. Each project focuses on using real-world datasets to build, train, and evaluate machine learning models using Python and popular libraries such as **Pandas**, **NumPy**, **Matplotlib**, **Seaborn**, and **Scikit-learn**.

---

## üìÇ Projects Overview

### 1. Bank Customer Churn Prediction
- **Objective:** Predict whether a bank customer will leave (churn) based on their demographic and account details.  
- **Techniques Used:** Logistic Regression, Random Forest Classifier, Data Encoding, Feature Scaling.  
- **Dataset:** Bank customer details including age, balance, tenure, and activity levels.  
- **Outcome:** Achieved accurate churn classification with effective feature importance interpretation.

---

### 2. Credit Card Fraud Detection
- **Objective:** Detect fraudulent transactions using imbalanced classification techniques.  
- **Techniques Used:** Random Forest, XGBoost, SMOTE (for data balancing).  
- **Dataset:** Credit card transaction records labeled as fraudulent or genuine.  
- **Outcome:** High performance on recall metric to minimize false negatives in fraud detection.

---

### 3. Indian Movies Prediction
- **Objective:** Predict audience ratings or box office success of Indian movies based on their attributes.  
- **Techniques Used:** Linear Regression, Feature Engineering, Correlation Analysis.  
- **Dataset:** Contains movie metadata such as genre, language, cast, and budget.  
- **Outcome:** Developed insights into features contributing to commercial and critical success.

---

### 4. Iris Flower Classification using Logistic Regression
- **Objective:** Classify iris flowers into species (*Setosa*, *Versicolor*, *Virginica*) based on their petal and sepal measurements.  
- **Techniques Used:** Logistic Regression, Data Visualization, Model Evaluation with Confusion Matrix.  
- **Dataset:** Famous Iris dataset by Fisher (UCI repository).  
- **Outcome:** Clean model achieving near-perfect accuracy with clear class separations.

---

### 5. Titanic Survival Analysis and Classification
- **Objective:** Predict survival of passengers aboard the Titanic using demographic and ticket data.  
- **Techniques Used:** Logistic Regression, Data Imputation, Exploratory Data Analysis (EDA).  
- **Dataset:** Kaggle Titanic dataset.  
- **Outcome:** Created interpretable model identifying critical survival factors such as gender and class.
